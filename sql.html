Table Creation

To create a table in SQL, we follow the syntax: 

CREATE TABLE table_name (
        column1 datatype [optional_constraint],
        column2 datatype [optional_constraint],
        ...
        columnN datatype [optional_constraint]
    );
Example: 

        CREATE TABLE employees (
    id INT,
    name VARCHAR(50),
    age INT,
    salary INT
);
Output: 

      SQL query successfully executed.

Inserting values

To insert the values in the table, follow the following syntax: 

 
        INSERT INTO table_name (column1, column2, ..., columnN)
VALUES
    (value1, value2, ..., valueN),
    (value1, value2, ..., valueN),
    ...
    (value1, value2, ..., valueN);
Example: 

      INSERT INTO employees (id, name, age, salary) VALUES
      (1, 'John Doe', 35, 50000),
      (2, 'Jane Smith', 28, 60000),
      (3, 'Michael Johnson', NULL, 70000),
      (4, 'Alice Johnson', 45, 75000),
      (5, 'Bob Brown', 22, 45000);
Output: 

        SQL query successfully executed.
Lets see the table. 

        | id | name            | age | salary   |
        |----|-----------------|-----|----------|
        | 1  | John Doe        | 35  | 50000.00 |
        | 2  | Jane Smith      | 28  | 60000.00 |
        | 3  | Michael Johnson | NULL| 70000.00 |
        | 4  | Alice Johnson   | 45  | 75000.00 |
        | 5  | Bob Brown       | 22  | 45000.00 |

Updating

For updating a table in SQL, follow the following syntax: Syntax: 


        UPDATE table_name
SET column1 = value1, column2 = value2, ..., columnN = valueN
[WHERE condition];
Example: 

        UPDATE employees
SET age = 36
WHERE name = 'John Doe';
Output: 

        SQL query successfully executed.

Deleting and Dropping

For deleting a table in SQL: 


          -- Deleting Rows from a Table
DELETE FROM employees WHERE age < 25;

-- Dropping a Table
DROP TABLE employees;

Querying Database Table

To query a table:


      -- Retrieve all employees
      SELECT * FROM employees;

Output:


      | id | name            | age | salary   |
      |----|-----------------|-----|----------|
      | 1  | John Doe        | 35  | 50000.00 |
      | 2  | Jane Smith      | 28  | 60000.00 |
      | 3  | Michael Johnson | NULL| 70000.00 |
      | 4  | Alice Johnson   | 45  | 75000.00 |

Aggregate Functions and Grouping

Example: 


    -- Calculating average salary of employees by age group
SELECT age, AVG(salary) AS avg_salary
FROM employees
GROUP BY age;
Output: 

    age | avg_salary
----------------
28  | 60000.00
35  | 50000.00
40  | 70000.00

Creation and Deletion of Views

Example: 


    -- Creating a view to display employees with salary greater than 60000
CREATE VIEW high_salary_employees AS
SELECT * FROM employees WHERE salary > 60000;

-- Deleting the created view
DROP VIEW high_salary_employees;

Output: 

  View high_salary_employees created.
View high_salary_employees deleted.

Grant and Revoke Commands

Example: 


    -- Granting SELECT privilege on the employees table to a user
GRANT SELECT ON employees TO user1;

-- Revoking INSERT privilege on the employees table from a user
REVOKE INSERT ON employees FROM user2;
Output: 

    SELECT privilege granted to user1 on employees table.
INSERT privilege revoked from user2 on employees table.

Commit and Rollback

Example: 


    -- Initiating a transaction
    BEGIN TRANSACTION;
    
    -- Performing SQL statements
    
    -- Committing the transaction
    COMMIT;
    
    -- Alternatively, rolling back the transaction
    ROLLBACK;
Output: 

    Transaction committed successfully.
Transaction rolled back.

Joins

Example: 


    -- Retrieving employee names along with their department names using INNER JOIN
    SELECT employees.name, departments.department_name
    FROM employees
    INNER JOIN departments ON employees.department_id = departments.id;
Output: 

    name           | department_name
-------------------------------
John Doe       | Sales
Jane Smith     | Marketing
Michael Johnson| HR

SET Operators

Example: 


    -- Retrieving a combined list of current and former employees using UNION
    SELECT * FROM employees
    UNION
    SELECT * FROM former_employees;
Output: 

    Combined list of current and former employees.
